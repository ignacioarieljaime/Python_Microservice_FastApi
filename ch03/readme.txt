=========================== Basic Details ==================================
Application Name: Online Recipe System
Developer: Sherwin John Calleja Tragura
Status: Prototype stage
Date finished: October 20, 2021

=========================== Description ====================================

This prototype aims to store various recipes on any menu to be viewed and 
rated by visitors. It even classifies the recipes according to the place 
of origin and menu type. Users may search for their desired recipe for 
reference, rating, and feedback purposes. It has a feature that allows 
users to attach keywords to each recipe for search purposes. 
The microservice application is an initial architecture for a robust 
online recipe platform. 

This application uses the FastAPI framework driven by Python 3.8.
No database included yet.

The requirements.txt will guide you with the dependencies. Just install by running:
pip install -r requirements.txt
